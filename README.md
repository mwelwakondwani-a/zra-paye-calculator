# ZRA PAYE Calculator
A simple web application to calculate PAYE for employees in Zambia.

#Overview 
This project allows users to input their monthly salary, allowances, and deductions, 
and calculates Gross Salary, PAYE, and Net Salary. It’s designed for educational purposes 
and small-scale payroll calculations in Zambia.
#Features
- Input monthly salary and allowances
- Calculates PAYE based on ZRA rates
- Displays Gross and Net Salary
- Simple and responsive interface
#Installation
1. Clone the repository: 
   git clone https://github.com/username/zra-paye-calculator.git
2. Open `index.html` in your browser.
3. No additional dependencies are requires.
#Usage
1. Enter your basic salary in the "Basic Salary" field.
2. Enter allowances in the "Allowances" field.
3. Enter deductions in the "Other Deductions" field.
4. Click "Calculate" to see Gross Salary, PAYE, and Net Salary.
#File structure#
- index.html — main HTML page
- style.css — styling for the application
- main.js — contains all JavaScript logic
